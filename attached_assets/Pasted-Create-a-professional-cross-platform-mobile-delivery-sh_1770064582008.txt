Create a professional cross-platform mobile delivery shopping application (Android & iOS) using a single codebase.

IMPORTANT:
- All app UI text must be in Arabic.
- Full Right-To-Left (RTL) support is required.
- App logic and code structure can be in English.
- The app must be easily developed and tested on Replit.

App Concept:
A local mobile delivery and shopping app that allows users to browse and order products from multiple essential categories within one city.

Technology Stack:
- Frontend: React Native (Expo)
- Backend: Node.js + Express
- Database: SQLite or JSON (lightweight, Replit-friendly)
- API-based architecture

Design & UI:
- Modern, clean mobile UI
- Color harmony inspired by the "Talabaty" app (inspiration only, no copying)
- Primary colors: soft orange, white, light gray
- Rounded cards, smooth spacing, modern icons
- Mobile-first design optimized for small screens

Main App Screens:

1. Splash Screen:
- App logo
- App name in Arabic

2. Home Screen:
- Search bar (Arabic placeholder)
- Banner slider for offers
- Main shopping categories:
  (مواد غذائية، سوبر ماركت، صيدلية، إلكترونيات، ملابس، خدمات)
- Featured products

3. Categories Screen:
- Grid view of all categories

4. Products Screen:
- Product card with:
  - Image
  - Name (Arabic)
  - Price
  - “إضافة إلى السلة” button

5. Cart Screen:
- Selected items list
- Quantity controls
- Automatic total calculation
- “إتمام الطلب” button

6. Checkout Screen:
- Customer name
- Phone number
- Address
- Order notes
- Order confirmation message in Arabic

7. Admin Panel (Basic - Web or Hidden Screen):
- Manage categories
- Manage products
- View orders

Functional Requirements:
- Smooth navigation between screens
- Arabic fonts with proper RTL alignment
- Local state management for cart
- API communication with backend

Future-Ready Features:
- Driver system
- Order status tracking
- Push notifications
- Online payment integration

Output Requirements:
- Complete runnable mobile app code
- Backend API code
- Instructions to run the project on Replit
- Instructions to preview the app using Expo Go
- No copyrighted assets or brand usage
